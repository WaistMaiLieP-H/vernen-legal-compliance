# Master Project Intake & Audit Triage

## Purpose

MANDATORY FIRST EXECUTION. Before analyzing any document or engaging any project, this skill triggers to map complete scope and build all required infrastructure.

## Trigger

ALWAYS FIRST on any new project, document, or entity.

## Execution

### Step 1: Entity Identification
### Step 2: Document Type Mapping per Entity
### Step 3: Governing Standards per Document Type
### Step 4: Required Audit Layers per Document Type
### Step 5: Skill Inventory — Exists or Gap
### Step 6: Create All Missing Skills Before Proceeding
### Step 7: Execution Sequence with Deadlines
### Step 8: Output Complete Project Map

## Rule

NO ANALYSIS UNTIL MAP COMPLETE AND ALL SKILLS EXIST.
