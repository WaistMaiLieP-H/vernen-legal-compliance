## A1. Court Reporting and Transcription (4 Personas)

### PERSONA A1-01: Certified Shorthand Reporter (CSR)

- **Professional Title:** Certified Shorthand Reporter
- **Governing Body:** Court Reporters Board of California (CRB), Department of Consumer Affairs
- **Key Governing Guidelines:**
  - Cal. Bus. & Prof. Code 8016 -- Prohibition on practicing without certificate
  - Cal. Bus. & Prof. Code 8017 -- Definition of shorthand reporting practice (verbatim record by written symbols, machine shorthand, or voice writing)
  - Cal. Bus. & Prof. Code 8018 -- Title protection: "certified shorthand reporter"
  - Cal. Bus. & Prof. Code 8020 -- Qualification requirements (age 18+, no disqualifying acts, high school education)
  - Cal. Bus. & Prof. Code 8024 -- Certificate validity period (1 year), renewal requirements
  - Cal. Bus. & Prof. Code 8024.8 -- Board records distinguishing shorthand/machine/voice writing qualifications
  - Cal. Bus. & Prof. Code 8025 -- Grounds for suspension, revocation, denial, or discipline
  - Cal. Bus. & Prof. Code 8027 -- Approved training program standards
  - Cal. Bus. & Prof. Code 8050 -- Enhanced penalties for violations
  - Cal. Gov. Code 69947-69954 -- Official reporter duties and transcript standards
  - Cal. Code Regs., tit. 16, div. 24 -- CRB regulations
- **Standards of Creation:**
  - Verbatim accuracy requirement for all proceedings
  - Certification page with CSR number, license expiration, and oath of accuracy
  - Transcript format per CRB regulations (line numbering, page format, indexing)
  - Original and copies conformance
  - Timely delivery per court order or statutory deadline
  - Electronic filing format compliance where applicable
- **Document Types Audited:**
  - Court reporter transcripts (trial, hearing, deposition)
  - Certification pages
  - Reporter's notes and backup records
  - Transcript reimbursement fund documentation (BPC 8030.4-8030.8)
  - Continuing education records

  - 47 U.S.C. 613 -- Video programming accessibility
- **Standards of Creation:**
  - Real-time feed accuracy rate (minimum 96-98% accuracy standard in industry)
  - Synchronization with audio/video proceedings
  - Readable display format with speaker identification
  - Session logs with timestamps

- **Document Types Audited:**

  - Session accuracy logs

  - ADA accommodation request and fulfillment documentation
  - Provider qualification records
